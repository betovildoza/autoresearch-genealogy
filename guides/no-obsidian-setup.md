# No-Obsidian Setup

You do not need Obsidian to use this kit. A normal folder and any text editor work.

## What You Need

- A folder on your computer.
- A markdown editor, notes app, or plain text editor.
- Your family notes and source files.
- Optional: an AI tool that can read and edit files.

No command line is required.

## Set Up The Folder

1. Download or clone this repository.
2. Copy the `vault-template/` folder.
3. Rename the copy to `Genealogy`.
4. Put it somewhere easy to find.

On Mac, use Finder.

On Windows, use File Explorer.

## Minimal File Set

Start with only these files:

- `Family_Tree.md`: people, relationships, dates, places, and source notes.
- `Research_Log.md`: every search you ran, including failed searches.
- `Open_Questions.md`: unknowns, contradictions, and next research targets.
- `Data_Inventory.md`: documents, photos, scans, DNA files, and where they live.

You can ignore the other template files until you need them.

## How To Open Files

Any of these work:

- Obsidian.
- VS Code.
- Typora.
- iA Writer.
- Notepad.
- TextEdit in plain-text mode.
- Any editor that can open `.md` files.

Markdown is plain text. The formatting marks are visible, but the files are still readable.

## What Wikilinks Mean

You may see links like this:

```markdown
[[Open_Questions]]
```

That means “look for a file called `Open_Questions.md`.”

If your editor does not make it clickable, use your folder search and open the file manually.

## Simple Folder Search

When a guide says “search the vault,” you can just search the folder:

- Mac: open Finder, go to your `Genealogy` folder, use the search box.
- Windows: open File Explorer, go to your `Genealogy` folder, use the search box.

Search for surnames, places, dates, record types, and source names.

## Safe First Workflow

1. Open `Family_Tree.md`.
2. Add only what you already know.
3. Mark living people as `Living`.
4. Open `Data_Inventory.md`.
5. List your documents and photos.
6. Open `Open_Questions.md`.
7. Add the top three unknowns.
8. Use [First Week Checklist](../checklists/first-week-checklist.md).
9. Use [Prompt Picker](prompt-picker.md).

## Using AI Without Obsidian

If your AI tool can read files:

- Point it at the `Genealogy` folder.
- Tell it to edit only files inside that folder.
- Ask it to log searches in `Research_Log.md`.

If your AI tool cannot read files:

- Paste only the relevant file sections.
- Remove living-person details first.
- Paste the prompt you want to run.
- Manually copy accepted results back into your files.

## When To Add More Files

Add more template files only when they solve a real problem:

- Use person files when one ancestor has lots of records.
- Use `Timeline.md` when dates are hard to compare.
- Use `Chromosome_Painting.md` when working with DNA segments.
- Use `Witness_Network.md` when names repeat across records.

Start small. A simple, sourced folder beats a complicated folder you stop using.
