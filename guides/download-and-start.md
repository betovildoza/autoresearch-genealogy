# Download And Start

Use this path if you do not use Git or the command line.

## Download

1. Open the GitHub repository page in your browser.
2. Select **Code**.
3. Select **Download ZIP**.
4. Unzip the download.
5. Open the folder named `autoresearch-genealogy`.

## Make Your Working Folder

1. Copy the `vault-template/` folder.
2. Rename the copy to `Genealogy`.
3. Move `Genealogy` somewhere easy to find, such as Documents.
4. Open [START_HERE.md](../START_HERE.md) in the downloaded repository folder.

Do not edit the original `vault-template/` folder. Keep it as a clean spare copy.

## First 10 Minutes

1. Open `Genealogy/Family_Tree.md`.
2. Add only facts you already know.
3. Mark living people as `Living`.
4. Open `Genealogy/Data_Inventory.md`.
5. List your documents, photos, and DNA files.
6. Open `Genealogy/Open_Questions.md`.
7. Add your top three questions.
8. Use [First Week Checklist](../checklists/first-week-checklist.md).
9. Use [Prompt Picker](prompt-picker.md).

## What Not To Do First

- Do not run tree expansion before privacy review.
- Do not paste exact living-person details into public AI tools.
- Do not add ancestors from copied online trees before checking sources.
- Do not upload raw DNA anywhere without a separate privacy and consent review.

## If You Want AI Help

If your AI tool can read files, point it at your `Genealogy` folder and use [Prompt Picker](prompt-picker.md).

If your AI tool cannot read files, paste only the relevant redacted sections and manually copy back changes after human review.
