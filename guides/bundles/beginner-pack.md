# Beginner Pack

Use this when you are starting with family stories, a small tree, or scattered notes.

## Who It Is For

- You have names, approximate dates, or family stories.
- You are not sure which records exist yet.
- You want a safe first pass before autonomous expansion.

## What To Prepare

- A copy of `vault-template/` renamed to your genealogy folder.
- `Family_Tree.md` with only what you already know.
- `Research_Log.md` ready for searches and negative results.
- `Open_Questions.md` with the biggest unknowns.
- Living and possibly living people marked as `Living`, `life_status: living`, or `life_status: unknown`.

## Order

1. Use `guides/privacy-mode.md`.
2. Use `checklists/first-week-checklist.md`.
3. Read [Getting Started](../../workflows/getting-started.md).
4. If you already have source notes, run [02 Cross-Reference Audit](../../prompts/02-cross-reference-audit.md).
5. Run [05 Source Citation Audit](../../prompts/05-source-citation-audit.md).
6. Run [01 Tree Expansion](../../prompts/01-tree-expansion.md) only after the first review pass.

## Stop And Review

Stop before tree expansion if:

- Living people are still exposed.
- More than half the tree is unsourced family memory.
- The same person may appear under two names.
- You do not know where the AI should write results.

## Human Review

Use these checks before accepting changes:

- `review-cards/02-cross-reference-audit.md`
- `review-cards/05-source-citation-audit.md`
- `review-cards/01-tree-expansion.md`
- `checklists/before-you-add-an-ancestor.md`

## Expected Outputs

- A safer working folder.
- Known facts separated from guesses.
- A list of missing sources.
- A smaller set of high-value expansion targets.
