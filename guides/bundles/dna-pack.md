# DNA Pack

Use this when you have DNA ethnicity estimates, shared matches, chromosome segment data, or a theory about an unknown ancestor.

## Who It Is For

- You want DNA to guide documentary research.
- You have segment data or match clusters.
- You want to avoid over-claiming what DNA proves.

## What To Prepare

- Exported match or segment data if your provider allows it.
- `Genetic_Profile.md` and `Chromosome_Painting.md` from the vault template.
- A documented tree for at least the closest known ancestors.
- A list of matches you can discuss without exposing living-person details.

## Order

1. Read [DNA Interpretation Guardrails](../../reference/dna-interpretation-guardrails.md).
2. Use `guides/plain-language/evidence-vs-clues.md`.
3. Run [12 DNA Chromosome Analysis](../../prompts/12-dna-chromosome-analysis.md) only on redacted or locally safe data.
4. Run [08 Open Question Resolution](../../prompts/08-open-question-resolution.md) for one documentary question suggested by the DNA lead.
5. Run [02 Cross-Reference Audit](../../prompts/02-cross-reference-audit.md) before accepting a relationship.

## Stop And Review

Stop if:

- The claim relies only on ethnicity percentages.
- The segment is small, isolated, or unphased.
- The match is living and has not consented to being included.
- Two different family branches could explain the same signal.

## Human Review

Use these checks before accepting changes:

- `review-cards/12-dna-chromosome-analysis.md`
- `review-cards/08-open-question-resolution.md`
- `review-cards/02-cross-reference-audit.md`
- `checklists/verify-an-ai-finding.md`

## Expected Outputs

- DNA leads separated from proven facts.
- Segment or match notes that preserve uncertainty.
- One or more documentary research questions.
- No ancestor assignment from ethnicity alone.
